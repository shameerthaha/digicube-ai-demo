# DigiCube AI Email Triage — Live Demo

**[→ Try the live demo](https://digicube-ai-demo.vercel.app)**

This is a live demo built by [DigiCube](https://digicube.ai) that showcases AI-powered customer email triage. Paste any customer email and within 3 seconds you get: an urgency classification (urgent/high/medium/low), the customer's sentiment and intent, a bullet-point list of the key issues raised, a concrete recommended action, and a ready-to-send professional reply draft — all generated by Claude AI. For small businesses and solo founders drowning in customer emails, this kind of automation can reclaim hours every week: instead of reading, categorising, and drafting replies manually, the AI handles the first pass instantly, so you only spend time on responses that need a human touch.

## What it demonstrates

- **Structured AI output** — Claude returns JSON with typed fields (category, priority, sentiment, key issues, reply draft) via the AI SDK's `generateObject`
- **Real-world business logic** — triage rules, draft reply tone, suggested next actions
- **Sub-4-second response times** — fast enough for live customer workflows
- **Clean, inspectable code** — a real pattern prospects can adapt for their stack

## Stack

- **Next.js 15** (App Router) — frontend + API routes
- **Anthropic Claude** via AI SDK (`generateObject` with Zod schema)
- **Tailwind CSS** — zero-config styling
- **Vercel** — deployment

## Running locally

```bash
git clone https://github.com/shameerthaha/digicube-ai-demo
cd digicube-ai-demo
npm install
cp .env.example .env.local
# Add your ANTHROPIC_API_KEY to .env.local
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Environment variables

| Variable | Required | Description |
|---|---|---|
| `ANTHROPIC_API_KEY` | Yes | Your Anthropic API key — get one at [console.anthropic.com](https://console.anthropic.com) |
